Descripción del proyecto

Seal Code es un editor de texto avanzado que permite la creación de IDEs personalizados alrededor suya. La idea principal del proyecto es la creción de un core potente que permita su fácil implementación a la hora de crear un IDE para cualquier lenguaje de programación. Además, sin necesidad de tener conocimientos de Java FX, cualquier usuario puede crear sus plantillas de autocompletado y resalto de sintaxis.

El core de Seal Code se comunica mediante eventos con la lógica que el programador quiera implementar, permitiendo así personalizar cada aspecto del entorno de desarrollo que se busque crear. Este enfoque lo que busca es minimizar la necesidad de entender como funciona internamente el software facilitando así su implementación.
